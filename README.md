# SkyEmpire

A mobile-first airline empire simulator built with Expo, React Native, TypeScript, and Expo Router.

## What is included

- Bottom tab navigation with Dashboard, Map, Airports, Fleet, and Routes
- Local game state persisted with AsyncStorage
- Live game loop with route progress, revenue, and costs
- Fleet purchasing and active route progress visualization
- Animated cash ticker and metric cards

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start Expo:
   ```bash
   npm start
   ```
3. Use Expo Go or an emulator for iOS/Android.
